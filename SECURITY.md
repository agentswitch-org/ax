# Security Policy

## Reporting a Vulnerability

Report vulnerabilities to **support@agentswitch.org** or via [GitHub private vulnerability reporting](https://github.com/agentswitch-org/ax/security/advisories/new).

Please include a description of the issue, steps to reproduce, and the potential impact. We aim to acknowledge reports within 48 hours.

## Supported Versions

The **latest release** is the only supported version. Apply updates to stay current.
